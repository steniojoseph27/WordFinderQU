# WordFinderQU
Ejecutar el Programa Principal
1- Abrir la Solución en Visual Studio: Inicia Visual Studio y abre la solución que contiene el proyecto WordFinderQU.

2- Compilar el Proyecto: Selecciona Build -> Build Solution (o Ctrl + Shift + B) para compilar el proyecto y asegurarte de que no hay errores de compilación.

3- Ejecutar el Programa: Puedes ejecutar el programa de dos maneras:

Usando el depurador de Visual Studio seleccionando Debug -> Start Debugging (o F5).
Sin depurador seleccionando Debug -> Start Without Debugging (o Ctrl + F5).
4- Interactuar con el Programa: Si el programa requiere entrada del usuario o argumentos de línea de comandos, proporciona los datos necesarios según las indicaciones del programa.
